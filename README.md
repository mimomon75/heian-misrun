# Heian Misrun / 平安御簾ラン

C言語とraylibで個人制作した、2人並走型のランゲームです。

障害物を避けながら走行距離を伸ばすゲームで、1人用モードと2人用モードを実装しました。

## 主な工夫

### 距離同期ジャンプ

1人用モードでは、入力時に2人が同時にジャンプするのではなく、それぞれが同じ走行距離へ到達したタイミングでジャンプします。

ジャンプ予定距離をキューで管理し、位置の異なる2人を1つの入力で操作できる仕組みにしました。

### 回避不能な配置を抑える障害物生成

障害物の種類に応じて最低間隔を変え、避けられない配置が発生しにくいよう調整しています。

足場として乗れる障害物、地上障害物、上下に動く障害物など、異なる性質の障害物を実装しました。

### 季節演出

走行距離に応じて、桜、葉、紅葉、雪がクロスフェードで移り変わります。

季節ごとに障害物とアイテムの出現傾向も変化します。

## 実装した機能

- 1人用・2人用モード
- 2段ジャンプ
- 当たり判定と足場着地処理
- 障害物・アイテムのランダム生成
- 速度アップアイテム
- 画面遷移フェード
- 衝突時の吹き飛び演出
- Top 3ランキングのローカル保存
- BGM・効果音

## コードの見どころ

- `JumpQ`: 距離同期ジャンプ用のキュー
- `EnforceSpawnGap`: 障害物の組み合わせに応じた最低間隔の調整
- `ResolvePlatform`: 足場への着地処理
- `InsertScore`: Top 3ランキングの更新
- `DesiredParticleIndex`: 距離に応じた季節の切り替え
